# Privacy Policy for Tennis Tracker

**Last Updated:** February 12, 2026

## Introduction

Tennis Tracker ("we", "our", or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our mobile application Tennis Tracker (the "App").

By using the App, you agree to the collection and use of information in accordance with this policy. If you do not agree with the terms of this privacy policy, please do not use the App.

---

## 1. Information We Collect

### 1.1 Personal Information
When you register for an account, we collect:
- **Email address** - Used for account authentication
- **Name** (first and last name)
- **Age**
- **Playing preferences** (dominant hand, tennis racket type)

### 1.2 Match Data
When you use the App to track tennis matches, we collect:
- Opponent names
- Match scores and results
- Match dates
- Court types (clay, hard)
- Match locations
- Match format preferences

### 1.3 Usage Data
We automatically collect:
- **Device information** - Device type, operating system version
- **App usage data** - Features used, app interactions
- **Notification preferences** - Daily reminder settings
- **Language preference** - English or Serbian

### 1.4 No Location Data
We do **NOT** collect precise GPS location data. Location information you enter (e.g., "Central Park Courts") is optional text input only.

---

## 2. How We Use Your Information

We use the collected information for:

- **Account Management** - Creating and maintaining your user account
- **Data Storage** - Saving your tennis match history and statistics
- **Statistics & Analytics** - Calculating your win/loss record, streaks, and performance metrics
- **Notifications** - Sending daily reminders (if enabled by you)
- **Personalization** - Displaying content in your preferred language
- **App Improvement** - Understanding how users interact with the App to improve features

---

## 3. Data Storage and Security

### 3.1 Cloud Storage
Your data is stored securely using **Google Firebase**:
- **Firebase Authentication** - Manages user accounts
- **Cloud Firestore** - Stores match data and user profiles
- All data is encrypted in transit (HTTPS/TLS)
- Data is stored in Firebase's secure data centers

### 3.2 Local Storage
We use **AsyncStorage** to:
- Keep you logged in between app sessions
- Cache your preferences for faster app performance

### 3.3 Security Measures
We implement industry-standard security measures:
- Encrypted data transmission
- Secure authentication protocols
- Firebase security rules to prevent unauthorized access
- No data is stored unencrypted on your device
